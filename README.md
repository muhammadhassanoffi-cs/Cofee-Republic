# Cofee-Republic
Coffee Republic Website
Project Overview
Coffee Republic is a professional and modern website developed for a café named Coffee Republic. The website aims to provide a seamless and interactive experience for users, allowing them to explore coffee and food products, learn about the café, and interact with its services.

Features
•	Home Page: Highlights featured products and provides quick navigation to other pages.
•	Coffee Page: Showcases a variety of coffee products with options to add to the cart.
•	Menu Page: Displays the complete menu including drinks and food items.
•	Food Page: Lists all available food items with details and add-to-cart functionality.
•	Add to Cart Page: Shows selected products, allows quantity updates, and removal of items.
•	About Us Page: Provides information about Coffee Republic’s vision, mission, and team.
•	Contact Us Page: Contains a contact form and café contact information.
•	Product Search Feature: Users can search for specific products directly from the product pages to quickly find what they need.
•	Responsive Design: Optimized for both desktop and mobile viewing.
________________________________________
Technologies Used
•	HTML5 – Structure and layout of pages.
•	CSS3 – Styling and responsive design.
•	JavaScript – Dynamic features such as add-to-cart functionality and search.
